# 项目说明
*本项目搜集了一些常用的代码小工具，代码都是以测试用例的形式呈现，兴趣使然，欢迎补充。*  

1. JSON转换实例
在`zx.json`包下面有使用`FastJson`、`Gason`、`OrgJson`以及`JsonLib`实现的JSON数据和数组、集合类、JavaBean、日期以及JSON文件互相转换的例子。  

2. 生成缩略图
在`zx.thumbnail`包下面有一张`origin.jpg`的待缩略的图片，启动测试用例会在您的桌面上生成一张该图片的缩略图。  

3. 生成二维码
在`zx.qrcode`包下面的测试类有两个例子，一个是生成普通二维码以及将二维码扫描解析结果的例子，另外一个则是生成带有图片LOGO的二维码的例子。
